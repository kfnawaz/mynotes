# Data Compass Confluence Markdown Pages

This package contains Markdown conversions of the Confluence wiki markup pages `01_*.wiki` through `21_*.wiki`.

Each `.md` file corresponds to the same-numbered `.wiki` source page from the v6 package.

Notes:
- Confluence headings, tables, lists, code blocks, quotes, panels, bold text, and monospace values were converted to standard Markdown where possible.
- These Markdown files are intended for review, editing, or import into tools that support Markdown.
